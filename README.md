# MOBI GADGETS (Accso)

A small static website showcasing device skins (mobile and laptop) and accessories.

## Overview

This project contains HTML, CSS, and JavaScript files for a simple storefront-style site called **MOBI GADGETS**. It is intended to be opened directly in a browser (no build step).

## Features

- Landing page with navigation (`index.html`).
- Sections for mobile and laptop skins.
- Device-specific pages and assets organized by brand.

## Quick Start

1. Open `index.html` in your browser.
2. Use the top navigation to visit Mobile or Laptop pages.

No server or dependencies are required — just a browser.

## Project Structure (important files)

- `index.html` — main landing page
- `home/` — styles and scripts used on the homepage (`mobi.css`, `mobi.js`)
- `phone/` — mobile skins pages and assets (brands under `phone/`)
- `laptop/` — laptop skins pages
- `public/` — (empty/for publishing)
- `zbin/` — sample or experimental files

Example: open `phone/mobi-phone.html` to view mobile skins.
